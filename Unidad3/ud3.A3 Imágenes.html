1. Diferencias entre la etiqueta title y alt.
        Title:  es un tag de código HTML que le permite darle un título a una página web.
        Alt: es un atributo HTML para un texto que describe una imagen.
    
        ¿Para que se utilizan los elementos map y area?
        Map: junto con img y area , crea un mapa de imagen.
        Area: La etiqueta area te permite definir un área dentro del mapa.
